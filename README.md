# Vector Array Processing (C++)

Ця програма на C++ генерує масив дійсних випадкових чисел, виконує різні операції з ним та зберігає результати у текстові та JSON файли.

## Функціональність програми
- Генерує масив випадкових чисел
- Записує масив у файл
- Об’єднує числа у групи
- Сортує масив за зростанням та спаданням
- Видаляє дублікати
- Обчислює різні суми
- Записує всі результати у JSON файл

## Файли в проєкті
- `Source.cpp` – код програми
- `json.hpp` – бібліотека для JSON
- `a.txt` – початковий масив
- `b.txt` – згрупований масив
- `data.json` – результати
- `README.md` – опис проєкту

## Як запустити
1. Клонувати репозиторій: `git clone https://github.com/andrianochkaq/vectorArray.git`
2. Перейти в папку: `cd vectorArray`
3. Відкрити `Source.cpp` у Visual Studio
4. Запустити програму, ввести кількість елементів і розмір групування

## Результати
- `a.txt` – початковий масив
- `b.txt` – суми груп
- `data.json` – усі обчислення
